# Adaptive Tracing Across Four Microservice Systems

This repository packages the local study artifacts produced for the adaptive tracing project across four systems:

- Train-Ticket
- Timescale OpenTelemetry Demo
- Spring Petclinic Microservices
- Grafana QuickPizza

It contains the final paper materials, intermediate cross-system reports, experiment results saved on the local machine, report generators, study notes, and the local system files changed to support the adaptive tracing experiments.

## Repository layout

- `reports/`
  - Final paper outputs and cross-system report outputs.
- `generators/`
  - Python generators used to build the final report and supporting cross-system reports.
- `docs/`
  - Setup notes and helper scripts used during the fourth-system QuickPizza work.
- `systems/train-ticket-python/`
  - Full local Train-Ticket Python workspace used for the study.
- `systems/timescale-otel-demo/`
  - Timescale study artifacts, reports, experiment outputs, adaptive scripts, notes, and locally changed files.
- `systems/spring-petclinic-microservices/`
  - Petclinic study artifacts, reports, experiment outputs, adaptive scripts, and local configuration files.
- `systems/quickpizza/`
  - QuickPizza study artifacts, reports, experiment outputs, adaptive scripts, notes, and local configuration/code changes.
